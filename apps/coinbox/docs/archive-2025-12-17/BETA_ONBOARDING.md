## What is Coin Box?

Coin Box is a comprehensive financial platform offering:
- **Peer-to-Peer Lending**: Create loan tickets and invest in others
- **Savings Jars**: Automated savings with customizable goals
- **P2P Crypto Trading**: Buy and sell Bitcoin, Ethereum, and USDT
- **API Access**: Programmatic access to all features
- **Bulk Operations**: Process multiple transactions efficiently
- **Webhooks**: Real-time event notifications

## Getting Started

### 1. Create Your Account

1. Visit [https://coinbox.com/beta/signup](https://coinbox.com/beta/signup)
2. Enter your invitation code (provided in your email)
3. Complete the registration form
4. Verify your email address

### 2. Complete Your Profile

1. Navigate to Settings > Profile
2. Add your personal information
3. Set up two-factor authentication (recommended)
4. Add a profile picture

### 3. Verify Your Identity

For full access to all features:
1. Go to Settings > Verification
2. Upload required documents:
   - Government-issued ID
   - Proof of address (utility bill, bank statement)
3. Wait for verification (usually 24-48 hours)

### 4. Add Funds to Your Wallet

1. Navigate to Wallet
2. Click "Deposit"
3. Choose your payment method:
   - Bank transfer
   - Card payment
   - Crypto deposit
4. Follow the instructions

## Key Features

### Creating a Loan Ticket

1. Go to Loans > Create Ticket
2. Enter loan amount (R100 - R5,000 for beta)
3. Select duration (7-90 days)
4. Set interest rate (15-25%)
5. Review and confirm

### Making an Investment

1. Browse available loan tickets
2. Click on a ticket to view details
3. Enter investment amount
4. Confirm investment
5. Track returns in your portfolio

### Using the API (Beta Feature)

1. Go to Developer > API Keys
2. Create a new API key
3. Copy your key (you won't see it again!)
4. Read the [API Documentation](/docs/api)
5. Start making requests

### Setting Up Webhooks

1. Go to Developer > Webhooks
2. Click "Add Webhook"
3. Enter your endpoint URL (must be HTTPS)
4. Select events to subscribe to
5. Save and test

## Beta Program Guidelines

### What We're Testing

- **Core Functionality**: All lending and investment features
- **API Access**: Complete API with 24 endpoints
- **Bulk Operations**: Batch processing capabilities
- **Webhooks**: Real-time event notifications
- **Performance**: System stability under load
- **User Experience**: Overall usability and design

### Beta Limits

During beta, the following limits apply:
- **Loan Amount**: Max R5,000 per loan
- **Investment Amount**: Max R10,000 total
- **Crypto Trading**: Max R5,000 per order
- **API Calls**: 100 requests/minute (Pro tier)
- **Bulk Operations**: Max 20 items per batch

These limits will be increased for full launch.
